Bienvenida

El Gestor Académico es una aplicación diseñada para administrar la información académica de una institución educativa.

En este sistema se gestionarán las entidades principales: Estudiantes, Asignaturas y Notas.

En esta fase se desarrolla la estructura visual y organizacional del sistema. En la siguiente fase se implementará la lógica y conexión entre módulos.
